Front-end

O que é Desenvolvimento Front-end.

O desenvolvimento web front-end é o desenvolvimento da interface gráfica do usuário de um site
por meio do uso de HTML, CSS e JavaScript para que os usuários possam visualizar e interagir com esse site. (Fonte: Wikipedia)

O que faz um Desenvolvedor Front-end.

O profissional desenvolvedor front-end é o responsável por         
colocar em prática, através de códigos, o design de um site ou interface.

Principais tecnologias utilizadas no Front End.

HTML, CSS, JavaScript, Bootstrap e React são algumas das principais tecnologias do front-end
